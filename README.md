# Real-time-object-detection

Real-time object detection trained 


## Dependencies

* OpenCv
* imutils

```
pip install -r requirements.txt
```

## Docker


```
docker build -t object-detection .
```


Run the project with pre-trained model :

```
bash RunDocker.sh
```
